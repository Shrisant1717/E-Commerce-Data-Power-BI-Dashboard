Global Superstore Sales Dashboard – Power BI
📊 Project Overview

The Global Superstore Sales Dashboard is an interactive Power BI report designed to analyze sales performance, profitability, shipping costs, and product distribution across different markets and regions. The dashboard provides business insights that help stakeholders identify high-performing categories, profitable markets, and operational trends.

🚀 Features
📈 Sales analysis by Product Category
🌍 Sales distribution across Markets
🚚 Shipping Mode performance analysis
🗺️ Country-wise and State-wise sales visualization
💰 KPI Cards for:
Total Sales
Total Profit
Total Quantity Sold
Total Shipping Cost
🔍 Interactive slicers for:
Category
Market
📌 Geographic coverage statistics:
Number of Countries
Number of States
Number of Cities
